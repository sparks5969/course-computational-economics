Part 3.2 — Object

Part 3.2. Object

With the blueprint, we next generate an animal object. Here we create the first animal. Its name is cat and weighs 8 lbs.

In [3]:
animal1 = Animal(name = 'cat', weight = 8.0)

You can access the variables (or embedded data) of the animal using the animal object and the ‘dot’ notation.

In [4]:
print("The name of the first animal is {}".format(animal1.name))
print("The weight of the first animal is {}".format(animal1.weight))
Out [4]:
The name of the first animal is Cat
The weight of the first animal is 8.0

Because we defined a method "introduce_me" for the animal class, we can simply call this method to get information.

In [5]:
animal1.introduce_me()
Out [5]:
Hello, let me introduce myself.
-------------------------------
My name is Cat
 My weight is 8.0 pounds
-------------------------------

As you see, method can interact with the attribute of the class. Let's define a new method eat() for the animal class.

In [1]:
import math as m

class Animal:

    def __init__(self, name = 'Animal Doe', weight = 0.0):
        self.name = name
        self.weight = weight
    def introduce_me(self):
        print("Hello, let me introduce myself")
        print("My name is {}".format(self.name))
        print("My weight is {} pounds".format(self.weight))
        print("-------------------------------\n")

    def eat(self, foodAmount = 0.0):
        """Method: eat(foodAmount=0)  translates food input 'foodAmount'
        into additional weight according to the  following formula:
        new_weight = old_weight + sqrt(foodAmount)"""
        self.weight += m.sqrt(foodAmount)  # the += is a pythonic way to say self.weight = self.weight + m.sqrt(foodAmount)

Now we let the animal eat, by calling the method eat() on the cat object.

In [2]:
animal1 = Animal(name = 'cat', weight = 8.0)
animal1.eat(9)
animal1.introduce_me()
Out [2]:
Hello, let me introduce myself
My name is cat
My weight is 11.0 pounds
-------------------------------

As you see, now the method 'eat' first take the argument 9 as input, and then updated the weight attributes according the the rule we defined.

Part 2.3 — User-Defined Functions

Part 2. Functions in Python

2.3. User-defined function

  •  In Python, a user-defined function's declaration begins with the keyword `def` and followed by the function name.
  •  The function may take arguments(s) as input within the opening and closing parentheses, just after the function name followed by a colon.
  •  After defining the function name and arguments(s) a block of program statement(s) start at the next line and these statement(s) must be indented.
  • You can give any name to a user defined function EXCEPT the Python keywords (such as print, if, vars, etc.)
  • To call a function, use the function name followed by parenthesis.
In [1]:
def even_or_odd(x):
    if (x%2 ==0):
        print('even')
    else:
        print('odd')

# call the function
even_or_odd(9)
even_or_odd(12)
Out [1]:
odd
even

Next let's look at another (a bit more complex) example. We wanted to define a function to calculate the area of a circle.

In [1]:
def area_circle(radius):
    pi = 3.1415926
    area = pi * radius **2
    print(area)

In this example, we defined a function area_circle to calculate the area of circles. From this example, we can look at the components of a function:

  • the first line def area_circle(radius): is known as the function header
  • the radius inside the parentheses is the function parameter. This parameter is used just like a variable of the function.
  • the variable pi and area are known as local variables. They only exist within the function.

Once we defined the function, we can call it to do some calculation for us. when we call the function, simply type in the function name, and feed it with the argument (in this example, radius is the only input variable). The argument will then be passed to the parameter of the function and do the calculation for us.

In [2]:
area_circle(12.5)
Out [2]:
490.87384375

In this example, the function does the calculate for us, but it does not return any value. To let the function return a value, use the return keyword inside the function.

In [1]:
def area_circle(radius):
    pi = 3.1415926
    area = pi * radius **2
    return area

# now call the function to do calculate, and store the result.
result = area_circle(12.5)
print(result)
Out [1]:
current unemployment rate is 15.38 percent.

In the above example, we defined two functions. The first function is labor_force, it takes population and participation as two inputs, and calculate the total labor force number. The second function is unemployment_rate, it take population, participation, and out_of_job as input variables, and calculate the unemployment rate. Note that inside the unemployment_rate function, we directly call the function labor_force to calculate the number of labor force. This is neat, as we don't have to repeat ourselves when calculating the unemployment rate.

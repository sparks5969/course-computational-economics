Installation of Python and Anaconda

The easiest way to install Python and most of the useful packages for scientific computing is to install Python via the Anaconda distribution from: Anaconda Individual EditionThis is a simple one-click installation process which works for Linux/Windows/Mac. Simply follow the instruction, it installs everything!

  • When you install anaconda and python, select the latest version Python for your system

  • For most of the computers, you wanted the 64 bit version, not the 32 bit version.

In order to write python script and let your computer run your script, you also need an Integrated development environment (IDE). With the Anaconda you just installed, open Anaconda-navigator by click its icon, and then install Spyder as your IDE. Starting from here, whenever you need to code, just open Anaconda-navigator, and then launch Spyder. 

There are many different Integrated development environment (IDE) for python, such as Spyder, VScode, Pycharm, etc. But unless you are VERY CONFIDENT about setting up and managing a virtual environment, please follow these simple steps and use Spyder as your IDE.

Basic Programming Techniques

5.1. Conditional Statements

In many situations, we many want to the script behave differently based on different conditions, we can do this with CONDITIONAL STATEMENTS in python. Oftentimes, we need to use comparison operators to evaluate if a condition is true or false. Like most of programming languages, python supports the usual logical conditions from mathematics. Below is a list of commonly used comparison operators in python. Please make sure you understand the meaning for each of them.

Operator Description
== Equal: condition is true if both operands are equal
!= Not equal: condition is true if operands are not equal
> Greater than: condition is true if left operand is greater than right
< Less than: condition is true if left operand is less than right
>= Greater than or equal to
<= Less than or equal to

Below are some examples:

a = 5

b = 10

a == b

Out [1]:
False

In [2]:
a = 5
b = 10
a != b

Out [2]:
True

The if command checks whether a certain condition is true. If it is true, we then can assign certain chain of commands under this condition. If the condition is not true, we can branch off the program into another direction and assign different commands. Based on this simple logic, we can branch the program into many separate possible directions. The keyword elif is similar, it is the python way to say, “if the previous conditions were not true, then try this condition”. Lastly, the keyword else catches anything which isn’t caught by the preceding conditions.

The general syntax of if statements are as follow:

if condition1:
    branch1
elif condition2:
    branch2
else:
    branch3

Now Let’s look at some specific examples:

In [1]:
current_price = 40
equilibrium_price = 50

if current_price < equilibrium_price:
    print('There will be a shortage on the market.')
elif current_price > equilibrium_price:
    print('There will be a surplus on the market.')
else:
    print('The market is clear.')

   

In the above example, I wanted to branch my script condition on the current market situation (i.e., the comparison of current price and equilibrium price). The if ... elif ... else ... statements captured all possible situations. When I run the script, I get:

Out [1]:
There will be a shortage on the market

One thing we need to pay special attention on is that PYTHON IS INDENT SENSITIVE! This is different from most of other programming languages. In python, indentation is a way telling python that the group of statement below belongs to a particular block of code. If you didn’t use indentation properly, you will encounter errors when running the script. 

In the above if-statements example, the branches are indented four spaces, which is a typical amount of indentation for Python. If I forgot the indentation rule, the code becomes:

In [2]:
current_price = 40
equilibrium_price = 50

if current_price < equilibrium_price:
print('There will be a shortage on the market.')
elif current_price > equilibrium_price:
print('There will be a surplus on the market.')
else:
print('The market is clear.')

 

In such case, python won’t be able to complete the task for you. It will generates an error message to help you correct the code:

Out [2]:
IndentationError: expected an indented block

I personally like the indent sensitive feature a lot, as it is an effective way to keep tracking the logic of my code. It also makes my code looks pretty and friendly to read.

5.3. Loops

There are several techniques you can use to repeatedly execute Python code. A loop is a structure that repeats a sequence of instructions until a specific condition is met. The loop structure is very useful when we wanted to repeat something over and over again. Two of the most common types of loops are while-loops and for-loops.

  • While-loops are like repeated if statements,

  • the for-loop iterates over all kinds of data structures.

5.3.1 While-Loop

The while loop executes a set of statements as long as a condition is true. Let’s look at some examples:

In [1]:
output = 10
equilibrium_quantity = 50

while output < equilibrium_quantity:
    output = output + 10
    print('current output level is %s.' %(output))

In this example, as long as current output is smaller than the equilibrium_quantity, we increase the output by 10 units and print out the current output level. Then we go back to the top of the while loop and check the condition again, until output level is not smaller than equilibrium_quantity. When run the above code, we get:

Out [1]:
current output level is 20.
current output level is 30.
current output level is 40.
current output level is 50.

5.3.2 For-Loop

A for loop is used for iterating over a sequence. The sequence can be either a list, a tuple, a dictionary, a set, or a string. Here is one example:Suppose we have a list of student name, we can print out the students in this list using for loop.

In [1]:
students = ['Mary','Lisa','Hope','Mike']
for x in students:
    print(x)
Out [1]:
Mary
Lisa
Hope
Mike

When combining the loops and conditional statements together, we can let python to deal with more complex tasks. For example, with the grade book we created earlier, we can let python find the students whose midterm grade is better than 90.

In [2]:
for student in gradebook:
    if gradebook[student][0]>=90:
        print(student)
Out [2]:
Lisa

Data Structures in Python

Data structures are a way of organizing and storing data so that they can be accessed and worked with efficiently. They define the relationship between the data, and the operations that can be performed on the data. There are many various kinds of data structures defined that make it easier for the data scientists and the computer engineers, alike to concentrate on the main picture of solving larger problems rather than getting lost in the details of data description and access.

Generally, data structures can be divided into two categories in computer science: primitive and collective data structures. The former are the simplest forms of representing data, whereas the latter are more advanced: they contain the primitive data structures within more complex data structures for special purposes. There are four types of primitive data in Python: Integer, float, Boolean, and String. Commonly used collective data structures include List, Tuple, Dictionary, and Set. Let’s look at each of them below.

4.1. Primitive Data Type

  • Integer represents numeric data, and more specifically, whole numbers from negative infinity to infinity, like 4, 5, or -1.

In [1]:
x = 1
y = 2
  • Float stands for ‘floating point number’. You can use it for rational numbers, usually ending with a decimal figure, such as 1.11 or 3.14. In [1]: 
x = 1.01
y = 2.04
  • Boolean is used to sort True and False values in python, and therefore, True and False are the only two boolean values.Remember, Python is case sensitive, so make sure capitalize the first letter when using Boolean values. In [1]: 
print(1 > 2)
Out [1]:
False
  • String is simply some characters inside quotes. Often times we use strings to represent words or phrases that need to be print out. In data analysis, we need to deal with string a lot. Python has a set of built-in methods that help us play around with strings. Let’s see some examples.

Suppose I have string “econ380: computational economics”, and I want to convert this string into upper cases. I can use the method .upper() to do this.

In [1]:
my_str = “computational economics”
print(my_str)
print(my_str.upper())
Out [1]:
computational economics
COMPUTATIONAL ECONOMICS

If I want to count how many times the letter “c” occurs in the string “computational economics”, I can use the method .count()

In [1]:
my_str = “computational economics”
my_str.count(“c”)
Out [1]:
3

There are many other useful string methods, you can [check hereLinks to an external site.] for a comprehensive list. But most likely, you will be learning more string methods and trick when you need to use them.

4.2. Collective Data Type

List, Tuple, and Set are all used to store multiple items in a single variable. But there are some important distinctions among them.

Notation Ordered Changeable Allow Duplicates Example
[ ] List Yes Yes Yes [human, bat, orc, human, elf, undead, orc]
( ) Tuple Yes No Yes (human, bat, orc, human, elf, undead, orc)
{ } Set No No No {elf, human, bat, orc, undead}

  • Ordered: if the items in it have a defined order;

  • Changeable: if we can change the items in it after it was created;

  • Allow duplicates: if it can hold two items with the same value.

Dictionary is used to store data values in {key:value} pairs within curly brackets. A dictionary is a collection which is ordered, changeable and does not allow duplicates in keys. That is to say, you can have as many pairs as you want, but keys in dictionary are unique.A dictionary in python looks like this: 

{“human”: “cavalier”, “bat”: “vampire”, “orc”:  “warrior”, “elf”: “archer”}. Visually, you can imagine the dictionary in table format:

Key Value
"human" "cavalier"
"bat" "vampire"
"orc" "warrior"
"elf" "archer"

4.2.1. python List

A Python list is an ordered and changeable collection of elements. It is very convenient and useful way to store information, especially when we are trying to handle many data points at once.

Suppose I wanted to record students’ midterm grade, I can create a variable for each record, such as:

In [1]:
grade1 = 87
grade2 = 93
grade3 = 75
grade4 = 66

As you can tell, creating a new python variable for each grade can be very inconvenient and inefficient. Instead, we can create a list and store all the values together. Let’s build a list with squared brackets:

In [1]:
grades = [87, 93, 75, 66]

Elements in list can be any type. In this example, we have integers in the list, but we can also have other types of element in the list, such as string, float, or Boolean. In more advance practices, it is also perfectly ok to have list contain sub-lists. You will see more examples and practices in our weekly project.

4.2.2. Python Dictionary

Python dictionary is another type data structure. It is best to think of a dictionary as a set of key: value pairs, with the requirement that the keys are unique (within one dictionary). A pair of braces creates an empty dictionary: { }. Placing a comma-separated list of key:value pairs within the braces adds initial key:value pairs to the dictionary; this is also the way dictionaries are written on output. It is a very useful tool for storing information. It stores the data values like a map. You can always look up information in a dictionary based on its key.

In our previous example, we saw how to use list to store student grade information. But one problem with the previous example is that I do not have student’s name associate with the grade. The problem is I won’t be able to know who gets which grade. In situations like this, we can use python dictionary to solve the problem.

Let’s create a dictionary with brackets, and then fill the dictionary with students’ names and their midterm and final grades.

In [1]:
gradebook = { } # create an empty dict named "gradebook"

# then fill the dictionary with students' names
# and their midterm grades + final grades + majors.

gradebook = {
'Mary' : [87,91,'economics'],
'Lisa' : [93,95,'computer science'],
'Hope' : [75,90,'psychology'],
'Mike' : [66,89,'biology']
}

In the above example, we created a python dictionary named “grades”. In this dictionary, the keys are students name, and the values are lists of [midterm, final, major]. To access elements in the dictionary, we can use the .get function associated with dictionary. This method will returns the value of the item with the specified key. For example, if I wanted to know Mike’s information, I can do the following:

In [1]:
gradebook.get(‘Mike’)

When I run the above line in the console, I can get the value associate with the key:

Out [1]:
[66, 89, ‘biology’]

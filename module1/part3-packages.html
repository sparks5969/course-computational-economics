Python Packages and Modules

Very simply, A module is a file containing Python definitions and statements intended for use in other Python programs. You can think of python packages as directory of python scripts. Each script is a module that contain certain functions, methods, and new types aim to solving particular problems. When a module from an external package is required in a program, that package can be imported and its modules can be put to use.

Think about it this way: you are a wizard who can use spells and charms do many different things. You can think of the “spells” as scripts in python. Oftentimes, scripts related with each other are put together as a ’Package”.

Whenever you need to use them, you can simply import them without knowing how the scripts are written. This is awesome, because it means we can use other people’s magic with simple incantations. Some of the packages are available when you first install python, but not all packages are available by default. In order to use those packages, you need to first install them on your own system. After you installed the package on your own system, you can then import and use it.

Here is one example: pandas is a package designed for data manipulation and analysis. We will use it a lot through this semester.

When you need to use functions in this package, simply tell python that I need this package. In your python script, type:

In [1]:
import pandas as pd

Now the pandas package has been imported into your python and ready to use! Let’s try to get our class roster into python.

We can use the .read_csv module in pandas to do this:

In [1]:
import pandas as pd
classmates = pd.read_csv('class_roster.csv')

Now we have the read our class roster into python. We saved it in a variable called “classmates”.

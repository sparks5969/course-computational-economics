1.2. Functions associated with python dictionary

Similar to list, a dictionary object also has functions associate with it. Let's look at some examples. 

There are several ways to create a dictionary, but the most straightforward way is to explicitly define the key:value pairs.

In[1]:

mycar = {
    "brand":"Subaru",
    "model":"Forester",
    "year":2005,
    "color":"green"
}
print(mycar)

Out[1]:

{'brand': 'Subaru', 'model': 'Forester', 'year': 2005, 'color': 'green'}

To access items in a dictionary, you can refer to its key name, inside square brackets:

In[2]:

x = mycar['color']
print(x)

Out[2]:

green

Alternatively, you can also call the .get() function to access the item in the dictionary:

In[3]:

x = mycar.get("color")
print(x)

Out[3]:

green

 

We can loop through a dictionary by using a for loop. When looping through a dictionary, the return value are the keys of the dictionary. But we can certainly figure out ways to return the values as well: the .values() function will return values of a dictionary.

In[3]:

for x in mycar:
    print(x)

Out[3]:

brand
model
year
color

In[4]:

for x in mycar.values():
    print(x)

Out[4]:

Subaru
Forester
2005
green

We can also loop through both keys and values by using the .items() function.

In[5]:

for x, y in mycar.items():
    print(x, y)

Out[5]:

brand Subaru
model Forester
year 2005
color green

 

To add an item to the dictionary, we can create a new index key and assigning a value to it:

In[6]:

mycar['miles'] = 72000
print(mycar)

Out[6]:

{'brand': 'Subaru', 'model': 'Forester', 'year': 2005, 'color': 'green', 'miles': 72000}

Alternatively, we can also call the .update() function associated with dictionary object to  insert the specific items.

In[7]:

mycar.update({'register':'Connecticut'})
print(mycar)

Out[7]:

{'brand': 'Subaru', 'model': 'Forester', 'year': 2005, 'color': 'green', 'miles': 72000, 'register': 'Connecticut'}

Part 4. JSON data format

In brief, JSON (JavaScript Object Notation) is one of the most popular ways for data storage and exchange, which is accomplished through its syntax, and is used in many web applications. It can be easily used to store and communicate information to other products.  Many programming languages, including python, support the JSON data format.

 

In python, there is a built-in package ‘json’ that can be used to read json file and convert a JSON string into a python object. Since JSON file store information based on key:value pairs, which is very similar to python dictionary It is the most convenient to convert JSON file to python dictionary. Here is an example.

 

In our weekly project, you will need to import a json file I provided and then start play with it. Here is how it works:

In[1]:

import json
# import the datasets
with open('project2_data.json') as f:
    preference = json.load(f)

The above block of code will load the 'project2_data.josn' file I provided, and store it in a python dictionary named as 'preference'. ======================================================================================================

 

Part 5. Comment your code

When writing your code in python, it’s important to make sure that your code can be easily understood by others and by yourself. Giving variables obvious names, defining explicit functions, and organizing your code are all great ways to do this. Another awesome and easy way to increase the readability of your code is by using comments. In addition, comments can also be used to prevent execution when testing the code.

 

Programmers forget what their own code does all the time, especially if the code was written a long time ago or under pressure. Writing comments as you go is a great way to keep your memory fresh. It will also push yourself to think about if your code make sense and keep improve it. Using proper comments throughout your code can also help others understand how it all works very quickly. This is very important when collaborating with others.

 

When you need to make a comment, simply put a hash mark # before the line you wanted to comment. Let’s look at this example:

In[1]:

# this line is a comment, python won't run it.
print('this sentence will be printed out in you terminal.')  # you can insert comments anywhere
# python ignores everything AFTER the hash mark and UP TO the end of the line.
# If you wanted to write a long comment that may take more than one line,
# just put a hash mark on each subsequent line.
print('Good luck with the weekly project.')
# now I think you got the idea. let's run this block of code and see what will happen.

Out[1]:

this sentence will be printed out in your terminal.
Good luck with the weekly project.

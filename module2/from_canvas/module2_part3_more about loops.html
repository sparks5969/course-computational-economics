Part 3. More about loops

In the previous week, we studied how to use loops to perform iterations in Python. In this section, we extend the discussion to nested loops (i.e., loops inside loops).  We will  learn about how to use the break and continue statements to control the loops behavior. The ultimatum goal is to master the magic of nested loops and use them to complete conditional, repeated tasks.

 

3.1. Nested loops

Let's take while loop as an example and look at the following code:

In[1]:

i, j = 1, 5
while i < 4:
    while j < 8:
        print("now running the inner loop")
        print(i, ",", j)
        j = j + 1   
    print("now running the outer loop") 
    print(i, ",", j)
    i = i+1

Out[1]:

    now running the inner loop
    1 , 5
    now running the inner loop
    1 , 6
    now running the inner loop
    1 , 7
    now running the outer loop
    1 , 8
    now running the outer loop
    2 , 8
    now running the outer loop
    3 , 8

 

In the above example, we see two while-loops nested together. The "outer loop" check the condition i<4 before every iteration, the "inner loop" check the condition j<8  before every iteration. When we run the block of code, here is what happened: 

  • We first assign two variable i=1  and j=5;  
  • Then python will start with check if i<4 (i.e., the outer loop condition) is True,
    • when i<4 is True, python will move forward to the inner loop, and check if the condition j<8 is True;
      • when j<8 is True, python will then execute the command inside the inner loop.
      • After the first iteration, we will have i=1, j=6
      • then python will go back to the top of the inner loop, and check the condition j<8 again. 
      • the inner iteration will keep running until j<8 become False
    • When the condition j<8 become False, python will jump out of the outer loop, and execute the commands below the inner loop.
    • When it reaches the bottom of the code, python will go back to the top of the outer loop, check the condition i<4 again, and determine if to start the next iteration.
  • The whole process ends when the condition i<4 become False.

 

1.2. Break and Continue

When we use loops to perform iteration, another useful skill is to stop the iteration when certain condition is met. Typically there are two ways to stop a running loop: the break statement and continue statement. 

 

The break statement is used to terminate the current iteration and exist the loop immediately. If the break statement is inside a nested loop, the break statement will terminate the innermost loop. Let's look at an example:

 

In[1]:

name_list = ['Mary','Diana','Dorothy','Jessica','Esther','Valerie',
'Brenda','Alfreda','Martha','Cari']
while name_list:    #This is a pythonic way to say "while name_list is not empty"
    name = name_list.pop(0)
    if name !='Dorothy':   
        print("current name is %s." %(name))
    else:
        print("Break the loop when see the name Dorothy")
        break  # terminate the loop
print(name_list)  #Let's see who are still in the name list

In this  example, we first create a name_list with 10 names in it. Then we use a while loop to pop the names and print them out one by one. During the iterations, we want python to stop the loop whenever the name "Dorothy" appears.  Below is the output when running the above bock of code. 

Out[1]:

current name is Mary.
current name is Diana.
Break the loop when see the name Dorothy.
['Jessica', 'Esther', 'Valerie', 'Brenda', 'Alfreda', 'Martha', 'Cari']

 

The continue statement are similar to the break statement. The continue statement is used to skip the rest of the code inside a loop ONLY for the current iteration. It will not terminate the loop, but continues on with the next iteration. Still using the above example:

In[1]:

name_list = ['Mary','Diana','Dorothy','Jessica','Esther','Valerie',
'Brenda','Alfreda','Martha','Cari']
while name_list:    #This is a pythonic way to say "while name_list is not empty"
    name = name_list.pop(0)
    if name !='Dorothy':   
        print("current name is %s." %(name))
    else:
        print("skip one")
        continue. # continue on to the next iteration
print(name_list)  #Let's see who are still in the name list

Out[1]:

current name is Mary.
current name is Diana.
skip one
current name is Jessica.
current name is Esther.
current name is Valerie.
current name is Brenda.
current name is Alfreda.
current name is Martha.
current name is Cari.

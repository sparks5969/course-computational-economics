Part 2. Logic Operators

As we have learned, conditions are nice way to make decisions by asking if something equals True or False. Sometimes, we may want to test several different conditions or even combine the conditions together to make a decision. Logical operators are used to combine conditional statements.

There are three commonly used logical operators, and, or, not. Let’s look at each of them.

  • When use and, it returns True if both statements are true
In [1]:
x = 5
(x > 3 and x < 10)
# the above two lines will returns True, because BOTH conditions are True.
Out [1]:
True
  • When use or, it returns true if one of the statements is true.
In [2]:
x = 5
(x > 3 or x < 4)
# the above two lines will returns True,
# because one of the conditions (x > 3) is True
Out [2]:
True

The not operator has two common use cases:

Case 1. it reverse the result. i.e., it returns False if the result is True

In [3]:
x = 5
not (x > 3 and x < 10)
# returns False, because it reverse the result inside the parentheses.
Out [3]:
False

Case 2. it can be used to check if a data structure (i.e., list, dictionary, set, tuple) is empty. If the data structure is empty, it returns true, otherwise it returns False.

In [4]:
empty_list = [ ]
not empty_list
# returns True, because there is no element inside the empty_list
Out [4]:
True
In [5]:
list1 = [1,2,3]
not list1
# returns False, because list1 has elements in it, it is not empty.
Out [5]:
False

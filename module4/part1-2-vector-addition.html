Part 1.2 — Vector Addition

1.2. Calculation with Vectors: Addition

When the two vectors have the same shape, we can perform basic calculations such as addition, subtraction, multiplication, and division. python will perform ELEMENT-BY-ELEMENT operations with the input vectors.

Let's start with adding two vectors together:

In [1]:
import numpy as np
vec1 = np.array([1,3,5,7,9])
vec2 = np.array([2,6,10,14,18])

print('The original vectors are')
print(vec1,vec2)
print('-------------------------')
print('The sum of the two vectors are')
print(vec1 + vec2)
Out [1]:
The original vectors are
[1 3 5 7 9] [ 2  6 10 14 18]
-------------------------
The sum of the two vectors are
[ 3  9 15 21 27]

We can also add a scalar (i.e., a constant number) to a vector. Always remember, when we are working with the ndarray objects, python will perform the element-to-element operation.

In [1]:
import numpy as np
vec1 = np.array([1,3,5,7,9])

print('The original vectors are:')
print(vec1,vec2)
print('-------------------------')
print('Now adding 5 to vec1, we have:')
print(vec1 + 5)
Out [1]:
The original vectors are:
[1 3 5 7 9] [ 2  6 10 14 18]
-------------------------
Now adding 5 to vec1, we have:
[ 6  8 10 12 14]

At this point, let's emphasize it one more time: list and numpy ndarrays looks similar, but they are two different data types. To further illustrate on this, point, let's see what will happen if we use python list to do calculations:

In [1]:
list1 = [1,3,5,7,9]
list2 = [2,6,10,14,18]
print(list1+list2)
Out [1]:
[1, 3, 5, 7, 9, 2, 6, 10, 14, 18]

When we try to add two lists together, instead of performing the element-to-element operation, python will simply concatenate the two lists together to generate a new, longer list. This isn't the "math" we want.

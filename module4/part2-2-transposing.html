Part 2.2 — Transposing

2.2. Transposing

The transpose of a matrix is a new matrix whose rows are the columns of the original. In python, we use the function transpose() to do this.

In [1]:
import numpy as np
A = np.array([[1,2],[3,4]])
A_trans = A.transpose()
print(A)
print("--------")
print(A_trans)
Out [1]:
[[1 2]
 [3 4]]
--------
[[1 3]
 [2 4]]

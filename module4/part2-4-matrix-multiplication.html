Part 2.4 — Matrix Multiplication

2.4. Calculation with Matrices: Multiplication

When multiplying two matrices A and B, we need to make sure that the number of columns of matrix A is the same as the number of rows of matrix B. That is to say, if A has dimension m x n, then B has to be a matrix of n x r, where r could be any positive integer. The outcome matrix should have dimension m x r. In python, we can use the function np.dot( ) to multiply matrix together. This can be done in one line of code:

In [1]:
import numpy as np
A = np.array([[1,2,3],[4,5,6]])
B = np.array([[1,-2],[3,0],[5,4]])
C = np.dot(A,B)

print(C)

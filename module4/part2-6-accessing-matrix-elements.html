Part 2.6 — Accessing Matrix Elements

2.5. Accessing Elements of a Matrix

If we wanted to kwon the dimensions of a given matrix, we can ask what is the shape of the matrix:

In [1]:
import numpy as np
A = np.array([[1,2,3],[4,5,6]])
A.shape
Out [1]:
(2,3)

To access elements in a matrix, we can specify the position of the element using squared brackets. Below are some examples

In [1]:
import numpy as np
A = np.array([[11,12,13,14],[21,22,23,24],[31,32,33,34]])
# Element row 2, column 3
print("A[1,2] = ", A[1,2])
print("---------------")
# If we want see ALL elements in the First row, we can do the following:
print("A[0,:] = ", A[0,:])
print("---------------")
# Similarly, if we wanted to see all ALL elements in the second column:
print("A[:,1] = ", A[:,1])
print("---------------")
# if we wanted to see All elements EXCEPT the first row:
print("A[1:,:] = ", A[1:,])

# ALWAYS REMEMBER: PYTHON START INDEXING FROM 0 !!!
Out [1]:
A[1,2] =  23
---------------
A[0,:] =  [11 12 13 14]
---------------
A[:,1] =  [12 22 32]
---------------
A[1:,:] =  [[21 22 23 24]
 [31 32 33 34]]

If we want to get a subset of a matrix, we can use the take( ) function to achieve it. Basically, we can take the columns or rows we want to keep, and assign them into a new matrix

In [1]:
import numpy as np
# define a matrix first
A = np.array([[11,12,13,14],[21,22,23,24],[31,32,33,34]])
print(A)
print("---------------")
# let's try to remove the second column from matrix A
B = A.take([0,2,3],axis=1)
print(B)
print("---------------")
# we can also remove the second row:
C = A.take([0,2],axis=0)
print(C)
Out [1]:
[[11 12 13 14]
 [21 22 23 24]
 [31 32 33 34]]
---------------
[[11 13 14]
 [21 23 24]
 [31 33 34]]
---------------
[[11 12 13 14]
 [31 32 33 34]]

Another useful practice is to replace elements in a matrix based on certain conditions. We can use the condition as a logical statement in combination with the choose() function. Suppose I wanted to replace all the elements that are larger than 30 with the number 5. We can do the following:

In [1]:
import numpy as np
# define a matrix first
A = np.array([[11,12,13,14],[21,22,23,24],[31,32,33,34]])
print(A)
print("---------------")
# let's replace elements that are larger than 30 with the number 5
B =(A>30).choose(A,5)
print(B)
Out [1]:
[[11 12 13 14]
 [21 22 23 24]
 [31 32 33 34]]
---------------
[[11 12 13 14]
 [21 22 23 24]
 [ 5  5  5  5]]

2.3. Calculation with Matrices: Addition

When adding two matrices together, we need add all the elements of each matrix "ELEMENT-BY-ELEMENT". Formally:

Then

 

We can only add two matrices that have the same dimensions. If matrix A is m x n, then matrix B has to be m x n as well. A numerical example is presented below:

In[1]:

import numpy as np
A = np.array([[1,2],[3,4]])
B = np.array([[5,6],[7,8]])
C = A + B
print(A)
print("--------")
print(B)
print("--------")
print(C)

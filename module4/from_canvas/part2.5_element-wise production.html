2.5. Calculation with Matrices: Element-wise product

Another type of matrix manipulation is the  element-wise product (also known as the Hadamard product). The element-wise product is a binary operation that takes two matrices of the same dimensions and produces another matrix of the same dimension as the operands where each element i, j is the product of elements i, j of the original two matrices. In python, the element-wise product can be obtained with:

import numpy as np
A = np.array([[1,2],[3,4],[5,6]])
B = np.array([[2,1],[0,1],[1,-1]])
C = A*B

print(C)

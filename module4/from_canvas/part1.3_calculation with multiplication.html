1.3. Calculations with vectors: Multiplication

Similar as addition, we can multiply a vector with a scalar (i.e., a constant number). The scalar is multiplied with ALL numbers of the vector. Usually we use the letter to denote the scalar. Formally:

This operation can be easily done with NumPy. Take a look at this example:

In[1]:

import numpy as np
vec1 = np.array([1,3,5,7,9])
constant = 2

print('The original vectors are')
print(vec1,vec2)
print('-------------------------')
print('Multiply vec1 with the constant, we have:') 
print(constant * vec2)

Out[1]:

The original vectors are
[1 3 5 7 9] [ 2  6 10 14 18]
-------------------------
Multiply vec1 with the constant, we have:
[ 4 12 20 28 36]

With NumPy, we can perform this operation in one line of code:

import numpy as np
vec1 = np.array([1,3,5,7,9])
vec2 = np.array([2,4,6,8,10])

print('The original vectors are')
print(vec1,vec2)
print('-------------------------')
print('The inner product of the two vectors are:') 
print(np.sum(vec1*vec2))

-------------------------
